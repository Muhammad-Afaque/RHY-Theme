# Defense WordPress Theme

A custom WordPress theme developed for displaying products with extensive customization options and modern UI components.

## Key Features

### Product Display
- Detailed single product pages with:
  - Image galleries with thumbnail navigation
  - Product specifications tables
  - Downloadable PDF generation
  - WooCommerce integration
  - Quote request functionality
  - Breadcrumb navigation

### Navigation & Layout
- Responsive mega menu system
- Mobile-friendly navigation
- Custom header and footer layouts 
- Breadcrumb navigation system
- Category/taxonomy based organization

### JavaScript Features
- Slick slider integration for:
  - Product galleries
  - Featured product carousels
  - Brand/logo showcases
- Custom form handlers
- AJAX cart functionality
- PDF download capabilities

### Additional Features
- ACF integration for custom fields
- SVG support
- Custom post type support
- WooCommerce compatibility
- Print/Download PDF functionality
- Quote request system

## Technical Details

### Requirements
- WordPress 5.0+
- PHP 7.4+
- WooCommerce (optional)
- Advanced Custom Fields Pro

### Structure
- Modular PHP templates
- Separated CSS/JS assets
- Custom post types for products
- Custom taxonomies for product organization

### Development
Built with:
- PHP
- JavaScript/jQuery
- CSS3
- WooCommerce hooks
- WordPress core functions

## Usage

1. Install as a WordPress theme
2. Configure WooCommerce settings
3. Set up required ACF fields
4. Add products and categories
5. Customize theme options as needed

For detailed documentation and support, please contact the theme developer.
